Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?
  • Using getElementById
  • Using getElementsByClassName
  • Using getElementsByTagName
  • Using querySelector
  • Using querySelectorAll
What is event delegation in the context of the DOM, and why is it useful?

It is an event-handling pattern that allows us to handle events at a higher level in the DOM tree other than the level where the event was first received.It's useful because instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.

How do you manipulate an element attributes and styles using DOM?
  • Style : Element.style.property="value"
  • Set atrribute:Element.setAttribute("atrributeName","attributeValue").
  • Get attribute: Element.getAttribute("attributeName")
